## 项目运行
* yarn install 如果没有响应或者很慢的话，可以使用 npm install
* npm run serve 

## 配置说明
### header 操作栏
header 操作栏的配置文件在 src/assets/scripts/config.js 中，包括：
* 有无称线
* 字号大小
* 默认颜色
* 代码主题风格

### 默认文本
默认文本内容在 src/assets/scripts/default-content.js 中
